# Hospital Management System with UML

# Project Description

This project is a Hospital Management System developed using Object-Oriented Programming (OOP) concepts. It manages patient records, appointments, and billing operations efficiently.

# Features

* Add and view patient details
* Book appointments with doctors
* Generate billing information
* Simple console-based interface

# OOP Concepts Used

* Encapsulation
* Inheritance
* Abstraction
* Polymorphism

# UML Diagrams Included

* Use Case Diagram
* Class Diagram
* Sequence Diagram
* Activity Diagram

# Technologies Used

* Java
* OOP Concepts
* UML Design (Draw.io / StarUML)

# SDG Goal

This project supports **SDG 3: Good Health and Well-being** by improving hospital management efficiency.

# How to Run

1. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
2. Compile the program:

   ```
   javac HospitalManagementSystem.java
   ```
3. Run the program:

   ```
   java HospitalManagementSystem
   ```
